## Workflow Loop

```mermaid
sequenceDiagram
  participant U as You
  participant G as GSD

  U->>G: /gsd:new-project
  G-->>U: PROJECT / REQUIREMENTS / ROADMAP / STATE

  U->>G: /gsd:discuss-phase N
  G-->>U: N-CONTEXT.md

  U->>G: /gsd:plan-phase N
  G-->>U: N-RESEARCH.md + N-*-PLAN.md

  U->>G: /gsd:execute-phase N
  G-->>U: commits + N-*-SUMMARY.md + N-VERIFICATION.md

  U->>G: /gsd:verify-work N
  G-->>U: UAT steps + fix plans (if needed)
```
